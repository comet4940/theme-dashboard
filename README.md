## ✨ Features

- **Live prices** via Yahoo Finance (delayed 15min, auto-refreshes every 5 min)
- **Theme Insights bar** — avg move, gainer/loser count, HOT/BULLISH/WEAK signal per theme
- **Per-ticker signals** — SURGE, MOVE, DROP badges based on % move
- **Filter & sort** — filter by theme, sort by top movers / worst movers / volume
- **AI Insights modal** — click any ticker for a Claude-powered analysis in context of its theme
- **Zero backend** — pure HTML/CSS/JS, fully static, works on any CDN or GitHub Pages

## 🚀 Deploy in 3 Steps

### Option A — GitHub Pages (Recommended, Free)

1. **Fork or clone this repo**
   ```bash
   git clone https://github.com/YOUR-USERNAME/themetrack.git
   cd themetrack
   ```

2. **Push to GitHub**
   ```bash
   git add .
   git commit -m "initial deploy"
   git push origin main
   ```

3. **Enable GitHub Pages**
   - Go to your repo → **Settings** → **Pages**
   - Under *Source*, select **GitHub Actions**
   - The workflow in `.github/workflows/deploy.yml` will auto-deploy on every push
   - Your site will be live at `https://YOUR-USERNAME.github.io/themetrack`

### Option B — Netlify Drop (Instant, no account needed)

1. Go to [netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop the entire `themetrack` folder
3. Get an instant public URL

### Option C — Vercel

```bash
npx vercel --prod
```

## 🔧 Customization

All theme and ticker data lives inside `index.html` in the `THEMES` array near the top of the `<script>` block:

```javascript
const THEMES = [
  {
    id: 'ai-infra',
    label: 'AI Infrastructure',
    icon: '⬡',
    tickers: ['AMD', 'NVDA', 'TSLA', ...],
    insight: "Your static theme description here..."
  },
  // Add more themes...
];
```

**To add a new theme:** Copy an existing theme object, give it a unique `id`, add your tickers, and add a CSS color variable in the `:root` block.

**To add a ticker name:** Add it to the `TICKER_NAMES` object.
